🌿 HabitBot — Telegram трекер привычек

Телеграм-бот для отслеживания ежедневных привычек с системой streak, статистикой и утренними напоминаниями.

✨ Возможности

- **Добавление привычек** — до 10 привычек на пользователя
- **Ежедневные отметки** — удобные кнопки прямо в чате
- **Streak система** — счётчик дней подряд с эмодзи-прогрессом (🌱 → 🔥 → 🏆)
- **Статистика** — общее количество выполнений и текущий streak по каждой привычке
- **Утренние напоминания** — бот сам напоминает о невыполненных привычках
- **Мультипользовательский** — каждый пользователь видит только свои привычки

🛠 Стек

- [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot) — основная библиотека
- [aiosqlite](https://github.com/omnilib/aiosqlite) — асинхронная работа с SQLite
- [APScheduler](https://github.com/agronholm/apscheduler) — планировщик напоминаний


🚀 Запуск локально

1. Клонируй репозиторий
   
```
bash
git clone https://github.com/твой-username/habitbot.git
cd habitbot
```

2. Установи зависимости

```
bash
pip install -r requirements.txt
```

3. Получи токен бота

Напиши (https://t.me/BotFather) в Telegram команду `/newbot` и следуй инструкциям.

4. Вставь токен в `bot.py`

```python
BOT_TOKEN = "1234567890:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
```

5. Запусти

```
bash
python bot.py
```

⏰ Настройка времени напоминаний

В файле `scheduler.py` измени:

```
python
REMINDER_HOUR = 9    # часы (UTC)
REMINDER_MINUTE = 0  # минуты
```

|    Город     | UTC offset |   Пример для 9:00   |
|--------------|------------|---------------------|
|    Москва    |   UTC+3    | `REMINDER_HOUR = 6` |
| Екатеринбург |   UTC+5    | `REMINDER_HOUR = 4` |
|  Новосибирск |   UTC+7    | `REMINDER_HOUR = 2` |
|  Владивосток |   UTC+10   | `REMINDER_HOUR = 23`|

📖 Команды бота

|      Команда      | Описание |
|-------------------|----------|
|     `/start`      | Приветствие и список команд |
| `/add <название>` | Добавить привычку |
|     `/list`       | Список всех привычек |
|     `/done`       | Отметить выполненные за сегодня |
|     `/stats`      | Статистика и текущий streak |
|     `/delete`     | Удалить привычку |

🔥 Система streak

| Эмодзи |  Streak   | Описание |
|--------|-----------|----------|
|  😴   |   0 дней  | Ещё не начал |
|  🌱   |  1–2 дня  | Хорошее начало |
|  🔥   | 3–6 дней  | Войдёт в привычку |
|  💪   | 7–29 дней | Настоящая привычка |
|  🏆   | 30+ дней  | Чемпион |
